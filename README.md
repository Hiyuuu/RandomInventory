# RandomInventory
企画用 一定時間ごとに全プレイヤーのインベントリと場所が一斉に交換されるプラグイン

# 使い方
  - ``RandomInventory.jar`` を ``plugins`` フォルダに投入する。
  - ``/reload`` - サーバーをリロードする
  - ``config.yml`` に ``ChangeTime: 60`` といった項目があり、数値を変えると何秒ごとにアクションが発生するかを設定することができる。(反映にプラグインのリロード必要)

# ライセンス
